📒 Notes App – Project Script
<br>
🚀 Overview:-<br>
The Notes App is a simple yet powerful web application designed to help users create, manage, and organize their notes efficiently. It provides a clean and user-friendly interface where users can quickly jot down ideas, edit them, and keep track of important information in one place.
<br>
🎯 Purpose:-<br>
The main goal of this project is to build a lightweight and responsive note-taking system that demonstrates core web development concepts such as CRUD operations (Create, Read, Update, Delete), state management, and user interaction.
<br>
✨ Features:-<br>
📝 Add Notes<br>
  ->Users can create new notes instantly by entering text and saving it.<br>
📖 View Notes<br>
  ->All saved notes are displayed in an organized manner for easy access.<br>
✏️ Edit Notes<br>
  ->Existing notes can be updated anytime to reflect changes.<br>
🗑️ Delete Notes<br>
  ->Users can remove notes that are no longer needed.<br>
💾 Persistent Storage<br>
  ->Notes are stored locally (e.g., Local Storage / Database), so they remain available even after refreshing the page.<br>
📱 Responsive Design<br>
  ->The app works seamlessly across different devices and screen sizes.<br>
  
🛠️ Technologies:-<br>
  ->React.<br>

⚙️ How It Works:-<br>
  -The user enters a note in the input field.<br>
  -When the user clicks the "Add" button, the note is saved.<br>
  -The saved note appears instantly in the notes list.<br>
  -Users can:<br>
    -Edit the note<br>
    -Delete the note<br>
